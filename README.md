# https://www.yuxuan.cloud
Yuxuan's personal website.

## Development
```
python3 -m venv pythonenv
source pythonenv/bin/activate
pip install pelican==3.7.1 markdown==2.6.8

cd www.yuxuan.cloud
```

## Creating a new site
```
mkdir newsite
cd newsite
pelican-quickstart
```
